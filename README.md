
# Credit Risk Prediction

This is an end-to-end Machine Learning project that predicts whether a loan applicant is high risk or low risk.

---
##  Live App
https://credit-risk-prediction-eabtrajyp6mrl4cqhnwudt.streamlit.app/

---
## Models Used
- Logistic Regression
- Random Forest
- XGBoost
---
## Tech Stack
- Python
- Streamlit
- Scikit-learn
- Pandas
- NumPy
---
## Features
- User input based prediction
- Real-time risk score
- Interactive UI
---
## Project Structure
- app.py → Streamlit app  
- credits_model.pkl → trained ML model  
- requirements.txt → dependencies  
---
## Objective
To build a machine learning model that can predict credit risk based on applicant details and deploy it using Streamlit.

---

##  Conclusion
The project successfully predicts credit risk using machine learning models, with Logistic Regression giving the best performance. The model is deployed as an interactive web application.
